    Filling Blanks in a Database-Style Table

    August 11, 2025
    VBA, Excel

    Take #1 #

    Sometimes when receiving data, someone tries to “tidy it up” by only entering values in the very first row, leaving the rest of the cells blank.

    Take #2 #

    Let’s see how to fill in these blanks quickly and efficiently.
    For the sake of practice, I’ve set up simplified scenario. I handled it using a Do..Loop statement.

    Take #3 #

    VBA code snippets

    Sub wordDBFill()

Do
If ActiveCell.value <> "" Then
    ActiveCell.Offset(1, 0).Activate
Else
    ActiveCell.value = ActiveCell.End(xlUp).value
    ActiveCell.Offset(1, 0).Activate
End If
Loop Until ActiveCell.Offset(0, 1).value = ""

End Sub
