## 3. 필드 설명
- `title` → 페이지 제목  
- `date` → 생성 날짜 (목차·정렬에 사용)  
- `weight` → 사이드바에서 순서 지정 (작을수록 위에 표시)  
- `draft` → `true`면 빌드에 포함 안 됨

## 4. Book 테마 특징
- 상단 제목(`# 첫 번째 포스트`)은 본문에도 나오고, 사이드바에도 자동 반영됩니다.
- 내부 링크는 `/경로/` 형식으로 쓰면 Hugo가 빌드 시 자동 처리합니다.
